# Literature Review

## 1. Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks

This paper introduced Retrieval-Augmented Generation (RAG), combining external retrieval systems with language generation models.

Key contributions:
- External memory integration
- Improved factual grounding
- Reduced hallucinations
- Dynamic knowledge retrieval

Research relevance:
This project uses RAG principles to improve contextual reasoning and memory-aware response generation.

---

## 2. Attention Is All You Need

This paper introduced the Transformer architecture and self-attention mechanisms.

Key contributions:
- Self-attention
- Parallel sequence processing
- Context-aware token relationships

Research relevance:
Transformers form the backbone of modern LLM systems.

---

## 3. Memory-Augmented Neural Networks

This work explored neural systems enhanced with external memory structures.

Key contributions:
- Long-term memory retention
- Contextual information storage
- Dynamic memory access

Research relevance:
Supports the idea of persistent contextual reasoning systems.

---

# Research Gap

Although modern LLMs demonstrate impressive reasoning abilities, they still face:
- contextual limitations
- short-term memory constraints
- hallucinations
- weak personalization

This project explores how contextual memory and retrieval pipelines can improve adaptive reasoning in AI systems.
